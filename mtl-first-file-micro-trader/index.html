Withdrawal logic

How profit leaves the bot later

  1. Bot starts each cycle with a £10 active trading pot.
  2. If a winning trade closes above £10, excess is skimmed into the vault.
  3. When the vault reaches your threshold, you manually withdraw later.
  4. Automatic withdrawals are not enabled in this proof build.
